This is the centralized repository for our Mythrir D&D 5e homebrew campaign ("Loss, Legacy, and Lament") — a post-apocalyptic technological-future setting. Treat it as a living wiki: characters, NPCs, factions, locations, items, and session history all need to stay consistent with each other over time.

Before adding new content:
- Check existing files for established facts (names, relationships, items, timeline) and don't contradict them.
- Match existing formatting conventions — e.g. character files use a Class/Race/Setting header block, a weapons section, a 5e Description / Visual Adaptation table for magic items, and (when relevant) a full image-generation prompt with Style, Character, Expression & Pose, Outfit, Weapons, Atmosphere, and Negative Prompt sections.
- Keep mechanical 5e details (stats, item properties, rules text) separate from narrative/visual flavor, as in the existing files.

Organization:
- Suggest and maintain folders by type as the repo grows (e.g. Characters/, NPCs/, Locations/, Items/, Sessions/) rather than letting everything sit loose in the root.
- Use clear, consistent file naming (e.g. character full name as the filename).

When generating new content (NPCs, lore, recaps, homebrew items, art prompts), match the tone and aesthetic already established: dark, gritty, tech-noir fantasy — not generic medieval high fantasy.

For character art prompts specifically, use the named visual style defined in [Mythrir - Gilded Lament Art Style Guide.md](Mythrir%20-%20Gilded%20Lament%20Art%20Style%20Guide.md) as the base layer (painterly portrait, gold-leaf banner background, shard/faceted fabric texture, muted indigo/gold/accent palette), then layer character-specific pose/outfit/weapon details on top, matching the pattern already used in each character's own "Final Image Generation Prompt" section.

If something I ask for conflicts with established lore or an existing file, flag the conflict rather than silently overwriting it.
